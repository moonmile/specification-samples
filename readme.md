# システム開発者のための仕様書の基本と仕組みのサンプル文書

秀和システム「システム開発者のための仕様書の基本と仕組み」のサンプル文書です。
仕様書や設計書を作成する際に参考にしてください。

- /テンプレート/* : テンプレートファイル
- /説明付き/* : 吹き出し等で説明入り

説明付きでは、吹き出しで簡単な説明が入っています。
本書と合わせて参考にしてください。
